Java Developer

Kod Gemisi olarak her seviyeden Java (Spring Boot/Framework) yazılım geliştirici pozisyonunda görev alacak çalışma arkadaşları arıyoruz.

İş tanımı:

Türkiye'nin en büyük sigorta firmalarından birinde, Spring Boot (Java) kullanarak büyük ölçekli projelere dahil olup gerçek yaşam problemlerine temiz kod prensiplerine uygun çözüm üretmek.

Bu süreçte yazılım geliştiricilerden beklenen belli başlı sorumluluklar şöyle;

  • Kurum içindeki kodlama standartlarını takip etmek, bunlara uygun kod yazmak.
  • Kendi açtığı pull requestleri (PR) deploy olana kadar takip etmek.
    • Yazıp tamamladığı çözümler için pull request (PR) açmak, PR'ın gözden geçirilmesi için ilgili kişilere haber vermek ve süreci takip etmek.
    • PR'a gelebilecek yorumları yanıtlamak, varsa kodda gerekli değişiklikleri yaparak PR'ı güncellemek.
    • PR'ın mergelenmesini sağlamak ve deployment'ı takip etmek.
  • Kendisine atanan işleri Jira'da ilerletmek ve log girmek.

Gereksinimler:

Her başlıkta beklenen bilgi seviyesi ve detay, adayın başvurduğu seviyeye (junior/mid/senior) göre değişir.
  • Java veya Kotlin dillerinden en az birini iyi derecede bilmek
  • Spring Boot veya Spring Framework ile proje yapmış olmak
  • OOP ve RESTful API kavramlarını bilmek
  • En azından kavramsal düzeyde SQL bilmek
  • Git kullanabiliyor olmak
  • Maven veya Gradle'dan en az birini temel seviyede kullanabilmek
  • Teknik makale okuyabilecek kadar İngilizce bilmek

Artılar:

  • Microservis mimarisi kullanılan bir projede çalışmış olmak
  • DDD ve hexagonal architecture kavramlarını bir projede kullanmış olmak
  • Gitflow bilgisi
  • Günlük olarak GNU/Linux işletim sistemlerinden birini kullanıyor olmak
  • Başka bir Java frameworkü ile çalışmış olmak
  • PL/SQL bilgisi

Avantajlarımız:

  • Kendini teknik olarak geliştirebileceğin, yeni teknolojiler kullanılan dinamik projelerde çalışma imkanı
  • Tüm dünyada kabul görmüş best-practice SDLC süreçlerini öğrenme ve uygulama şansı
  • Şirket içi eğitimlere katılma imkanı
  • Yemek ve yol yardımı

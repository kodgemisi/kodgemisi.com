Git Eğitimi

Eğitim Hakkında

  • Eğitim kapsamında git kullanımı, git’in diğer sürüm takip sistemlerine göre avantajları ve proje kod tabanınıngit ile nasıl daha iyi yönetilebileceği anlatılacaktır.
  • Eğitim sonunda kullanıcıların git sürüm takip sistemini projelerinde kullanabilmesini amaçlar.
  • Eğitime mevcut projelerini git’e geçirmek isteyenler veya git’i daha verimli kullanmak isteyenler katılabilir.
